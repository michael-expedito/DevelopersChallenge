Bem vindo ao gerenciador de torneios!

Times

Como todo torneio possui times participantes, é no cadastro de "Times" que informaremos todos os times possíveis que farão parte dos torneios que o sistema irá gerenciar. Para o time podemos informar o seu nome com até 50 caracteres e uma descrição do time podendo ter até 150 caracteres.

Torneios

Depois de incluirmos no sistema todos os times que vão participar dos torneios, já podemos criar uma competição no sistema utilizando o cadastro do "Torneio". Aqui que definimos todos os times que farão parte do torneio definindo quem joga contra quem na fase inicial do torneio. O sistema possui uma funcionalidade de escolher aleatoriamente todos os times que estão cadastrados no sistema facilitando a criação do torneio. Nesse sistema um torneio pode possuir 2, 4, 8, 16 e até 32 times participantes.

Resultados

Depois que o torneio estiver cadastrado, todas as partidas que forem realizadas com seus respectivos vencedores deverão ser informadas no processo de "Resultados".

Partidas

Na rotina "Partidas" é permitido consultar de forma hierárquica todo o histórico de vencedores de um torneio.
